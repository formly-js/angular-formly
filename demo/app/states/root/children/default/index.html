Welcome to the official Angular Formly website! Feel free to stick around here and browse/play with the examples. If you want to see the docs, go to the angular-formly repo on GitHub.

Features

  • Prebuilt templates for and {{::prebuilt.name}}, and WIP template libraries for and {{::prebuilt.name}},
  • Simpler than cake do it yourself templates through the formlyConfig service.
  • Totally customizable fields (specify model for the form or for a specific field)
  • Template Wrappers make it very easy reduce the amount of boilerplate in your template code so you can maintain a consistent style without running into a maintainence nightmare
  • Template manipulators which allow you to modify templates programmatically. This comes with a built-in template manipulator which will automatically add attributes like `ng-required` or `ng-blur` when the field's config specifies such a requirement which reduces the cruft in your templates as well as the number of watchers for your forms considerably.
  • Take advantage of angular's validators api, whether you're using 1.2 or 1.3, with formly's super simple validators api with simple string expressions, advanced functions, and async validation support (in both 1.2 and 1.3).
  • expressionProperties makes it super easy to manipulate the fields as the user interacts with your app
  • Warnings logged and errors thrown to ensure you're using formly correctly. Links provided so you can resolve the issue quickly. We'll help you be effective with angular-formly :-D
  • If you're using Angular 1.3, then you can take advantage of the ng-model-options directive via modelOptions.
  • defaultOptions for field types, allowing for as DRY code as possible.
  • Custom link and controller functions per type and/or field, giving maximum control, without having to create a custom directive to add functionality.
  • Custom watchers with an enhanced API give you total control when you need it
