Welcome to the official Angular Formly website! Feel free to stick around here and browse/play with the examples. If you want to see the docs, go to the angular-formly repo on GitHub.

Features

  • Prebuilt templates for Bootstrap, LumX, and vanilla html
  • Do it yourself templates through the formlyConfigProvider
  • Totally customizable fields (specify model for the form or for a specific field)
  • Take advantage of angular's validators api, whether you're using 1.2 or 1.3, with formly's super simple validators api
  • expressionProperties makes it super easy to manipulate the fields as the user interacts with your app
  • Take advantage of modelOptions if you're using Angular 1.3 (but we support 1.2 as well)
  • Custom watchers with an enhanced API give you total control when you need it
